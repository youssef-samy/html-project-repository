Input and output devices act as the user-computer interface. A device which sends information to a processing computer system is called an input device and a device which receives and then reproduces or shows the results of the processing is called an output device. Many devices are either input devices or output devices, as they can only accept user input data And forward the same to your computer device, or receive computer output data. There are certain devices that can accept both input and output handling, these device types are called as I / O devices (input / output devices). Peripherals are also called input-output devices. We 're going to discuss some information about different input output devices.

Input Devices

-Input devices can send data or information to a computer or another device.

-Input devices cannot receive data or signal from some other devices.

Various input devices are as listed below:

  1. Keyboard: It is an input device which sends data in to the computer. The data send depends on the key pressed by the user.
  2. Mouse:A mouse is a small handheld input device which controls a cursor in a graphical user interface. It can move and select text, files, folders etc. on our computer according to the user input.
  3. Scanner:Optically, the scanner reads and documents, files or images, then transforms them into digital signals, and sends them to the computer.
  4. Webcam:it sends the captured images to a computer.
  5. Microphone:it receives audio generated by some input source and sends the same to a computer.
  6. Barcode reader:It is used to read the barcode of various items and feed the same to computer.
  7. OMR:optical mark recognition/ reader, is used to read marks on a document and send them to computer.
  8. MICR:Magnetic Ink Character Reader is an input device which generally finds application is banks to process cheques.
  9. Joystick:these input devices are used to control video games.
  10. OCR:OCR stands for optical character Recognition, is an input device which reads printed text and sends that to computer.
  11. Trackballs:trackball is an upside down mouse which is encased within a socket. It is a cursor control device.

Output Devices

A device capable of receiving data from a computer or other system and generating output with that data is called the output unit. Examples of various output devices are as follows:

  1. Monitor:A monitor is an output device that is responsible for receiving data from a computer and displaying that information as text or images for users to see on a screen.

    2. Types of Monitors

    • Cathode-ray tube (CRT):use picture tube technology; inexpensive, but they take up desk space and use a lot of energy.
    • Liquid Crystal Display (LCD or flat-panel):Cells sandwiched between two transparent layers form images; expensive, and they take up less desk space and use less energy than CRT monitors.
    • Light-emitting diode (LED):is more energy-efficient and has better color accuracy and thinner panels than LCD monitors.
  2. Speakers:Receives sound signal from a computer and then plays that sound signal and thus we hear songs or music or any other audio.
  3. Projector:Gets data from a computer and displays or projects the same information onto a screen or a wall. Projector cannot directly accept data from a user and send that data to another device.
  4. Plotter:is a printer that uses a pen that moves over a large revolving sheet of paper,It is used in engineering and map making.
  5. Printers:is a peripheral device that produces a physical copy or hard copy of the computer's output.

Both Input / Output Devices

An input/output device may receive data from users or other devices, and send data to other devices or computers as well. That means a devices which can be used as both input device and output device are called Input / Output (I/O) devices.Few examples of devices input / output are as follows:

  1. Touch Screen:Touch screen is both input and output device. By touching the screen input is provided and being a screen, it is used as an output device.
  2. USB drive:Often known as pen drive or flash stick , it functions both as a computer-input tool and as a tool for output. USB drives accept or save data as an input from a computer, and may also transfer data to a computer or another device.
  3. Headsets:Headset consists of speaker as an output device and microphone functions as an input device.
  4. CD-RW drives and DVD-RW drives:Receives data from a device as an input for copying to a writable CD or DVD and saving to that. We do use CDs or DVDs for the data transfer to a computer.
  5. Modems:It is used to transmit and receive data via telephone lines from one computer to another computer or to other devices.
  6. Facsimile:Facsimile or FAX machine requires a scanner to provide data and is an input tool and a small printer.
