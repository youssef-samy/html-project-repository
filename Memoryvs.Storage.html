What Is Memory

-Memory, as the link between your computer's central processing unit ( CPU) and data, is the most important component of a computing system.

-Memory consists of main memory and cache.

The main memory includes two types:
  • RAM (Random Access Memory)is volatile memory.
  • ROM (Read Only Memory) is non-volatile memory.

    *RAM

  • RAM is also called primary memory, which exchanges data with theCPU directly.
  • RAM is volatile. The data stored in the RAM will not stay when you shut down your computer. When you need to store such data as words and images, you need to save it in a long-term storage unit, like a hard drive.
  • It can be read and written anytime with a high speed. Usually, it runs much faster than almost all access devices, and its access latency is negligible compared with other storage devices involving mechanical operation, such as hard disks and CD drives.
  • RAM data comes from storage devices. If you want to access a file, it will be copied from storage devices and then you will be able to look at and edit it in the memory. If you don't save it again, the changes you've just made will be lost when the computer is shut off.

    *ROM

  • ROM is used to store software, sometimes known as firmware, which is rarely changed during system life.
  • Non-volatile memory (ROM) is read-only. The data stored in ROM, written by the memory system manufacture and then not modifiable, is stable and can not be lost except when the machine is turned off.

What Is Storage

-Storage refers to the permanent holding place for digital data, which is also called the secondary storage.

-It is not accessible directly through the CPU. Usually the device uses its input / output channels to access secondary storage and transfer to main memory the desired data. Storage is a server that can hold data without power, and data can be stored indefinitely in storage devices until it is intentionally deleted.

-The secondary storage include internal hard drives and external storage devices like USB, CD, etc.

-The internal hard drives have two types: HDD and SSD.Nowadays, many users like to upgrade their internal hard drives to SSDs.

Why is storage necessary?

Storage:

  • Is cheaper than memory.
  • Plays an input role when starting applications.
  • Is needed for output.
  • Plays an important role during startup.
  • Devices can hold a large amount of data.
  • Retains data when the computer is turned off.

Capacity and Speed of Storage Devices

A storage device's performance is measured by:

  • Capacity:The number of bytes of data that a device can hold.
  • Access Time:The amount of time, in milliseconds (ms), it takes for the device to begin reading data.
